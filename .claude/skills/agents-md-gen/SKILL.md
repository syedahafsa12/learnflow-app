# AGENTS.md Generator

## When to Use
- User asks to generate AGENTS.md for a repository
- Need to create documentation for AI agents about project structure
- New repository needs agent-friendly documentation

## Instructions
1. Run the generator script: `python scripts/generate_agents_md.py`
2. The script will analyze the repository structure and create AGENTS.md
3. Review and customize the generated file as needed

## Validation
- [ ] AGENTS.md file is created
- [ ] File contains repository structure information
- [ ] Includes file types and purposes

See [REFERENCE.md](./REFERENCE.md) for customization options.
